Unified Menu — Mockups

Plug in a controller and press any button — d-pad / left stick to move, A to select, B to back (Chrome Gamepad API, standard mapping). Keyboard works too: ↑↓←→, Enter (A), Esc (B). Mouse is intentionally not wired up.

  • 01 — Root menu (Game / Configuration / Quit)
  • 02a — Game picker, card row (chosen variant)
  • 02b — Game picker, list + art pane (rejected, kept for reference)
  • 03 — Game page (per-game art, New/Load/Save/Multiplayer)
  • 04 — Configuration (unified, tabbed, glyph style picker)
  • 05 — Configuration / Advanced tab (curated engine settings)
